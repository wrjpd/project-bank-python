# Projeto 3 - Bank

Projeto do curso **Programação do básico ao avançado**

## Descrição

Devemos desenvolver uma aplicação onde ao ser inicializada solicite ao usuário escolher o que deseja fazer no banco:

- criar uma conta
- efetuar saque
- efetuar depósito
- efetuar transferência
- listar contas
- sair do sistema
